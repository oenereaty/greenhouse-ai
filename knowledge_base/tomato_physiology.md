# 토마토 작물생리학 핵심 지식베이스

## 1. 온도 관련 임계값

### 착과 실패 조건
- 낮 기온 35°C 초과: 꽃가루 발아율 급감, 착과 실패
- 낮 기온 40°C 초과: 꽃가루 완전 불활성화
- 밤 기온 25°C 초과 3일 연속: 수정 불량, 낙과 증가
- 밤 기온 13°C 미만: 저온 장해, 기형과 발생

### 생육 최적 온도
- 낮 기온: 20~25°C
- 밤 기온: 13~17°C
- 지온: 20~23°C

### 생육 한계 온도
- 최저: 10°C (생육 정지)
- 최고: 35°C (고온 장해 시작)

## 2. VPD (증기압포차) 관리

| VPD (kPa) | 상태 | 작물 반응 |
|---|---|---|
| 0.4 미만 | 과습 | 증산 억제, 잿빛곰팡이 위험 |
| 0.4~0.8 | 낮음 | 생육 다소 불량 |
| 0.8~1.2 | 최적 | 이상적 증산·광합성 |
| 1.2~1.5 | 높음 | 관수 주의 |
| 1.5~2.0 | 스트레스 | 기공 폐쇄 시작 |
| 2.0 초과 | 위험 | 심각한 수분 스트레스 |

## 3. GDD (생육도일) 기준 (중생종 기준, Tbase = 10°C)

| 생육 단계 | 정식 후 누적 GDD |
|---|---|
| 정식 초기 | 0~100 |
| 영양생장 | 100~300 |
| 1화방 개화 | 300~450 |
| 착과 | 450~700 |
| 과실 비대 | 700~1000 |
| 수확 시작 | 1000 이상 |

> 품종에 따라 ±20% 차이 발생. 조생종은 약 800 GDD에 수확 시작.

## 4. 주요 병해 발생 조건

### 잿빛곰팡이(Botrytis cinerea)
- 발생 온도: 15~25°C (최적 20°C)
- 발생 습도: 90% 이상
- 조건: 잎 습윤 4시간 이상 지속
- 예방: 야간 환기, 상대습도 85% 이하 유지

### 흰가루병(Oidium neolycopersici)
- 발생 온도: 20~25°C
- 발생 습도: 50~70% (건조 조건에서 발생)
- 특이사항: 고온 건조 시 오히려 발생 증가
- 예방: 황 훈증, 규산 엽면 살포

### 역병(Phytophthora infestans)
- 발생 온도: 15~25°C
- 발생 습도: 85% 이상
- 조건: 잎 습윤 + 토양 과습
- 예방: 배수 관리, 하부 잎 제거

## 5. 관수 관리

- 기준: VPD 및 일사량 기반 적산 증산량으로 결정
- 고온기(VPD > 1.5): 소량 다회 관수
- 저온기(VPD < 0.6): 관수 억제, 과습 주의
- EC 목표: 2.5~3.5 dS/m (과실 비대기 상향)

## 6. 탄산가스(CO₂) 관리

- 최적 농도: 800~1000 ppm
- 환기 시 외기 농도(약 420 ppm)로 희석됨
- 여름철 환기 개방 → CO₂ 농도 저하 → 광합성 효율 감소
- 권장: 환기 최소화 시간대에 CO₂ 시비

## 7. 수확 적기 판단

- 착색 기준: 품종에 따라 붉은색 50~80% 발현 시 수확
- 가용성 고형물(당도): 4.5~6.0 Brix 이상
- 경도: 과육이 단단하고 탄력 있을 때
