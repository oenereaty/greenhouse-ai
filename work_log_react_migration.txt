남은 작업:
- JSON 파일 저장은 아직 파일 락/원자적 쓰기가 완전히 적용되지 않았다. 동시 요청 안정성을 위해 SQLite 또는 atomic write 유틸로 옮기는 작업이 필요하다.
- FastAPI 앱 시작 시 RAG 벡터스토어를 동기 로딩하는 구조는 여전히 부팅 지연 위험이 있다. lazy loading 또는 background warmup으로 바꾸는 것이 좋다.
- Streamlit 제거 여부가 확정되면 requirements.txt와 README에서 Streamlit 실행 경로를 정리해야 한다.
