===================================================================
  토마토 온실 AI 의사결정 지원 시스템 — 프로젝트 요약
===================================================================

■ 이 프로젝트는 무엇인가

한국 온실 토마토 농가를 위한 AI 의사결정 지원 도구입니다.
Streamlit 대시보드 위에서, 로컬에서 돌아가는 LLM(Ollama gemma4:12b)과
RAG(BGE-M3 임베딩 + ChromaDB), 그리고 여러 공공데이터 API를 결합해
"지금 온실 상태가 괜찮은지, 뭘 해야 하는지"를 실시간으로 알려줍니다.


■ 왜 의미가 있는가

1. 전부 로컬 실행 — 클라우드 LLM API 비용이 들지 않고, 센서·생육
   데이터가 외부로 나가지 않습니다. (ANTHROPIC_API_KEY, OPENAI_API_KEY
   불필요)

2. 근거 기반 답변(RAG) — 농진청 재배 매뉴얼, 병해충 방제목록,
   작물영양학(다량원소·미량원소·수분·양분흡수) 자료를 실제로 검색해서
   답변에 인용합니다. 모델이 아무 말이나 지어내지 않도록
   "참고자료에서 확인 안 되면 그렇게 명시하라"는 규칙을 프롬프트에
   못박아 뒀습니다.

3. 공공 데이터 통합 — 기상청(단기예보/AWS 실시간관측), aT 공판장
   경락가격, NCPMS 병해충 도감, 네이버 뉴스를 한 화면에서 조회합니다.

4. 그냥 챗봇이 아니라 에이전트 — AI 상담 탭은 MCP(Model Context
   Protocol) 에이전트로 동작해서, 필요하면 스스로 "지금 센서값 조회",
   "구역별 생육 데이터 비교" 같은 도구를 호출해 실제 데이터를 확인한
   뒤 답합니다.


■ 핵심 기능 (탭 구성)

1) 환경 데이터
   - 실시간 센서(온도·습도·CO2·일사) + VPD 자동 계산
   - AI 기반 이상 상황 감지 및 알림(경보)

2) 생육 데이터
   - 구역별 생육 측정값(초장·엽수·착과수 등) 기록/조회

3) 기상 데이터
   - 기상청 단기예보(3일) + AWS 실시간 관측 연동

4) 가격 정보
   - 경매현황: aT 공판장 실시간 경락가 원장 + 최소/평균/최대
   - 경매비교: 시장(익산·전주 등) 간 가격 추이 비교
   - 시장 동향 브리핑(신규): 네이버 뉴스에서 자재비·인건비·기상 관련
     기사를 모아 가격 변동 요인을 AI가 분석·요약 (기사 출처 명시)

5) 온실 제어
   - 제어 설정 및 관련 정보

6) 방제 기록
   - 병해충 예찰(규칙 기반 위험도 판정) + 방제 이력 관리

7) AI 상담
   - RAG + MCP 에이전트 통합 채팅 — 하나의 창에서 문서 검색과
     실시간 도구 호출(센서/생육/진단이력 조회)을 동시에 처리
   - 이미지 업로드 병해 진단 — 사진을 올리면 병해충 방제목록 전체를
     근거로 진단명·특징·방제법·전문가 확인 필요 경고를 안내
     (※ 시각적으로 애매한 병해는 오진 가능성이 있어, 실제 방제 전
     반드시 농업기술센터 확인 필요)

8) 메모 · 일지
   - 영농일지: 날짜별 캘린더, 방제/작업 기록, 수확 목표일 D-day
   - 파일 첨부(신규): 일지 기록에 사진 등 파일 첨부
   - 다가오는 계획 AI 알림(신규): 영농일지에 적어둔 미래 계획을
     감지해서, 버튼 클릭 시 지금부터 준비할 일을 체크리스트로 안내
   - 양액 조성 기록 + AI 분석(신규): 날짜별 양액 조성(N/P/K/Ca/Mg/
     EC/pH)과 관찰 증상을 기록하고, 작물영양학 자료를 근거로
     성분 부족/과잉 여부와 조성 비율 조정안을 AI가 제안


■ 기술 스택

- LLM: Ollama gemma4:12b (로컬, 텍스트+이미지 멀티모달)
- 임베딩: BGE-M3 (로컬 CPU 실행)
- 벡터 DB: ChromaDB (지식베이스 검색용)
- 에이전트: FastMCP 서버 + Ollama 도구 호출(Function Calling)
- 프레임워크: Streamlit
- 외부 API: 기상청 API Hub / data.go.kr, aT(농넷) 경락가격,
  NCPMS 병해충 도감, 네이버 뉴스 검색 API


■ 알아두면 좋은 한계

- 병해 이미지 진단은 범용 비전 모델 기반이라, 궤양병처럼 시각적으로
  판별이 까다로운 병해는 오진할 수 있습니다(실제 테스트로 확인됨).
  최종 방제는 반드시 전문가 확인을 거치도록 안내 문구가 붙어 있습니다.
- 이 앱은 "현재 센서값 기반 반응형 판단 보조 도구"이며, 미래 예측
  모델이 아닙니다. 자동제어 적용 전 현장 확인이 필요합니다.
